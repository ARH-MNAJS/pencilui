# @pencilui/react

React components for pencilui — hand-drawn / pencil-sketch UI library.

Wraps Radix UI primitives for behavior (a11y, keyboard, focus management) and overlays the visual layer from `@pencilui/styles`. Components are added stage-by-stage per the implementation plan.

`@pencilui/react/sketches` exposes inline-SVG sketch components used by hand-drawn motion (e.g. `CheckSketch` for the hand-checked checkbox). Borders, fills, and paper textures live in CSS, not here.
